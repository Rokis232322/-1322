# Генератор случайных паролей (Password Generator)

Графическое приложение на Python для генерации
криптографически стойких (в учебных целях — случайных) паролей
с возможностью сохранения истории в формате JSON.

## Автор

Бабичев Руслан Денисович

## Описание программы

Приложение позволяет:
- Задавать длину пароля (от 6 до 128 символов).
- Выбирать типы символов: заглавные буквы, цифры, специальные символы.
- Генерировать случайный пароль на основе выбранных настроек.
- Сохранять сгенерированный пароль с указанием произвольного имени.
- Просматривать историю сохранённых паролей в таблице.

Данные (список паролей) сохраняются в файле `storage.json`
в корневой папке программы и загружаются при следующем запуске.
Предусмотрена валидация настроек и имени записи.

## Требования

- Python 3.8 или выше
- Стандартная библиотека tkinter (входит в большинство
  дистрибутивов Python, на Linux может потребоваться
  установка пакета `python3-tk`)
